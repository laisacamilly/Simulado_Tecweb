>Plataforma de acompanhamento de ativos da B3, desenvolvida com React.

<h3> 🟣 Sobre o projeto </h3>

O **KYRA** é uma aplicação web para acompanhamento de ações brasileiras (B3). O usuário pode **buscar ativos pelo nome ou símbolo**, visualizar o preço atual, acessar um **resumo detalhado** com gráfico histórico, **favoritar ativos** e **enviar resumos por email.**

<h3> 🟣 Link do Deploy </h3>
https://projeto-2-frontend-la.onrender.com

---

<h3> 🟣 Funcionalidades </h3>

- 🔍 **Busca de ativos** — Pesquise por nome (ex: "Petrobras") ou símbolo (ex: "PETR4").
- 📊 **Resumo do ativo** — Ao clicar em um cartão, abre um popup com gráfico histórico, preço atual, variação, máxima, mínima, volume e horário de atualização.
- ⭐ **Favoritar/Desfavoritar** — Salve seus ativos favoritos e acesse-os na aba Favoritos.
- 🔥 **Trendings** — Veja os ativos em destaque na B3.
- 📧 **Enviar por email** — Envie um resumo do ativo para um email.
  
---

<h3> 🟣 Estrutura de telas </h3>

| Tela | Rota | Descrição |
|---|---|---|
| Início | `/` | Busca de ativos + Trendings |
| Favoritos | `/favoritos` | Ativos favoritados |
| Trendings | `/destaques` | Lista completa de destaques |
| Send Email | `/email` | Envio de resumo por email |

---

<h3> 🟣 Tecnologias </h3>

- [React](https://react.dev/) + [Vite](https://vitejs.dev/)
- React Router DOM
- CSS Modules
- Fetch API
